# Plan - Affaire Epstein Timeline

## Vue d'ensemble
Création d'une timeline immersive permettant de comparer :
- **La réalité judiciaire** (faits, dates exactes, documents officiels)
- **La couverture médiatique** (quand les médias ont parlé vs silence)
